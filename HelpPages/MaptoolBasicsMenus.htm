Maptool Basics: Menus

Here I'm going to go over the menus ("File", "Edit", and so on) and just touch on the stuff you need to know. I'll have a bunch of pictures here; usually, if you click on them, a larger version will open in another tab so you can see more easily.

File

There are several entries in the file menu you should know about.
Open Campaign
This is the first thing you should do after starting a server. It loads the actual campaign (maps, minis, macros, etc). But I usually use “Recent Campaigns” instead.
Save Campaign
This is the last thing you should do before turning off the server. It saves the campaign.
Save Campaign As
This is just like Save Campaign, only you can change the campign's name when you save it.
Save Chat Log History
This allows you to save the chat log.
Start Server
This is what you use to start a server. Hopefully you will never need to do that and either Scott or I will handle it. If you do need it, consult Maptool Basics: Starting a Server.
Recent Campaigns
If you go down here, a menu will open to the right with the recent campaigns you have loaded. This is what I use instead of open campaign because I am lazy.

Edit

There's not much you really need here.
Undo Drawing/Redo Drawing
You can undo or redo a drawing you made.
Clear Drawing
This removes all drawings on a given map layer (Token, Hidden, Object, or Background). The layer to be cleared is the one you've selected in the layer list in the top right corner of the map.
Copy/Cut/Paste
Once you've selected a token or tokens, you can copy/cut/paste them with these. Ctrl+C/Ctrl+X/Ctrl+V works as normal.
Edit Preferences
Important: Below are two of the four tabs for preferences. You want to make sure that all the ones I've circled in red are the same as in the examples below. It's probably a good idea to make sure the other settings are the same as mine, too, but you must make sure the ones in red are the same.

Map

This is a lot of stuff related to the map. You should make sure you understand this.
New Map
This creates a new map. I've gone over this in Making Maps.
Import Map and Export Map
Export Map saves an entire map (tokens and all) to a file, and Import Map loads one. After I make a map, I export it, then I import it and make sure it works, and if it does, I delete the map and the imported copy. This makes the campaign file smaller and more manageable.
Player Visible
If this is not checked, players cannot switch to this map.
Fog of War
Important: If this is not checked, the map will not have fog of war.
Vision
Important: these are the vision settings for the map:
Off
Vision is unrestricted. Everyone can see everything, regardless of VBL. This is bad.
Day
Vision is restricted by VBL, but everywhere on the map has light. Use this for outside during the day.
Night
Vision is restricted by VBL, and there is no atmospheric light, so tokens can only see things if they have darkvision or a light source. Use this for underground or indoors, or outside at night.
Edit Map
You can use this to change the map in various ways. It does exactly the same thing as New Map, only it changes the current map rather than making a new one.
Adjust Grid
This opens a little thing in the top right corner of the map which you can use to change the size of the grid, or offset it in relation to tokens which are not snapped to the grid, or to change the color of the grid lines if you care about that sort of thing.
Adjust Map Board
I never use this, but if you do maps the way I do, you can adjust the position of the map background image. I'm not totally sure how this works, to be honest.
Rename, Copy Map, Delete
These all seem obvious to me: rename a map, create a copy of the map, or delete the map.

View

I basically ignore this entirely. Still, a few useful bits and pieces:
Show as Player
If you click this and then select a token, you can see what that token sees. Sometimes useful. I mostly use it for testing VBL as I make it.
Zoom
Zooming in or out is super helpful, particularly as a GM. I use my mouse wheel, but you can use this, too.
Show Movement Distances
I always have this checked, and so should you. When a token is moved, you can see how far.
Show Grid
If you want to see the grid, check this.

Tool

A few useful bits and pieces here, too.
Center Players on Current View
This moves the players' view to the coordinates you're looking at. If people can't find their minis, it is sometimes helpful for you to find them and then center on them.
Enforce Player to Current Map
This brings the players' view to the map you're looking at.
Center Players on Current View (Continuous)
Please make sure not to check this.

Window

This is how you open a panel you might not already have open. I describe the panels I use in Maptool Basics: Maptool Panels.
