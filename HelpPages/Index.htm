Table of Contents

I've put together a bunch of webpages to help people deal with DMing. It's not all finished, but this is the landing page. I've broken these up into three sections.

Maptool Basics:

This describes the basics of using Maptool as a GM. There's a lot here. Don't panic. It won't take long to get a handle on. But be sure you read all of these before you take over as DM.
  • Making Maps
    This describes how to get maps into Maptool and how to handle vision, fog of war, and VBL.
  • Making Minis
    This describes how to make tokens in Maptool.
  • Maptool Panels
    This describes how to use the Campaign Panel, the Initiative Panel, and the Map Explorer Panel. All three will be very useful to you.
  • Menus
    This goes over the menus at the top ("File," "Edit," and so on).
  • Starting a Server
    This goes over starting a server. I should warn you that this is the part I am least able to troubleshoot.
  • Tools
    Finally, this goes over the various little icons underneath the menus at the top. You'll need this to handle VBL and Fog of War properly.

Rules Basics:

This is an attempt to provide the basic D&D rules you will need in an accessible form.
  • Combat
    This is a long one, but it goes over the very basic rules for what you can and cannot do in combat. I cover starting a combat, what kind of actions you can take in a combat, attacks of opportunity, and some important miscellaneous rules.
  • Skill and Ability Checks
    This covers all the important rules for making skill checks and ability checks.
  • Reading Statblocks
    This works through a couple of examples on reading a monster’s stat block. It’s pretty easy, but I figure, might as well be complete.

Tips & Tricks

This provides several GMing tips I've learned over the years.
  • Adventure Preparation
    This describes the prep work I do before we start an adventure.
  • Encounter Creation
    If you write your own adventure, you'll have to make your own encounters. If you use a pre-generated adventure, you might want to modify those encounters to reflect the fact that we are five instead of four. This describes how I do that.
  • Running Combat
    This short little page describes how I run a combat.
  • Setting the DC
    If you want the PCs to face a skill challenge of some kind, you need to decide on the DC of the skill or ability check. This is advice for how to do that.
